# Hairstyling Salon Website

This is the code repository for a hairstyling salon website built using Nuxt.js. The website features a 
modern design, responsive layout, and integration with Firebase and Stripe for backend services and payment processing.

## Nuxt environment variables

```bash
NUXT_SITE_URL=

NUXT_SITE_NAME=

NUXT_SITE_ENV="staging"

NUXT_REDIS_HOST=

# Firebase

NUXT_FIREBASE_API_KEY=...

NUXT_FIREBASE_AUTH_DOMAIN=...

NUXT_FIREBASE_DB_URL=...

NUXT_FIREBASE_STORAGE_BUCKET=...

NUXT_FIREBASE_APP_ID=...

NUXT_FIREBASE_MEASUREMENT_ID=...

NUXT_FIREBASE_MESSAGE_SENDER_ID=...

NUXT_FIREBASE_PROJECT_ID=...

# Stripe

NUXT_STRIPE_TEST_SECRET_KEY=...

NUXT_STRIPE_TEST_PUBLISHABLE_KEY=...

# Stripe

NUXT_REDIS_HOST=...

NUXT_REDIS_PASSWORD=...
```
